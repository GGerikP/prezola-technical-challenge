Simple Gift Registry

Welcome to this simple gift registry. This has been built by Gerik Peterson as a simple demo product for the Wedding Shop.

The following are the key features of this site:

  • Python Backend with a very simple frontend (if you couldn't tell..)
  • Django Middleware to manage the core of the application (and provide most of the system framework)
  • JSON API - Ensure we extract the front end from the backend as much as possible.
  • Two internal apps: the gift_registry_api responsible for the JSON API and a gift_registry_web responsible for serving up the simple web template pages
  • Simple relational sqlite3 database with a handful of models
  • Functional data management (via this simple GUI and the administration screens) - includes status changes and stock management.
  • Report display
  • The source code is stored on my github account where you'll also find the README file with more details.
